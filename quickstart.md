# Quickstart Guide

## Python Set up
brew install python3

## Ollama set up
1. Open a terminal
2. Run brew install ollama 
3. Run ollama serve
4. Open a second terminal
5. Run ollama pull qwen3:8b

To run ollama: ollama run qwen3:8b

## Entering/Exiting the virtual environment
Enter: source venv/bin/activate 
- should show (venv) before your name in terminal
Exit: deactivate 

Tip: You don't need to leave venv until you're done working on the project for the day/work session. You can still run git commands and test everything.
